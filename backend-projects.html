Back-End PHP Projects

CRUD Project with Custom-Build Framework

Functionality:

  • -Register
  • -Login
  • -Edit your own profile
  • -View all registered users
  • -Logout

Additional Information:

  • -Project has sessions and implements authentication.
  • -You can't view the site content unless you're not logged in.
  • -Data is stored in database (mySQL) via PDO connection.
  • -The data itself is transfered to other layers with DTO objects.
  • -Build using MVC architecture and OOP.
